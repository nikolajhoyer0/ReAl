
ReAl has keybinds for most of its menu items and buttons. Below is an overview:

Buttons:
Ctrl + Enter: Run
Ctrl + N: New sheet
Ctrl + R: Remove sheet
Ctrl + U: Save current table
Ctrl + T: Show tree

Menus:
Ctrl + S: Save project
Ctrl + O: Load project
Ctrl + K: Save script
Ctrl + L: Load script
Ctrl + I: Import table
Ctrl + E: Export table
Ctrl + D: Delete table
Ctrl + Q: Quit
F1: About
F2: Help
